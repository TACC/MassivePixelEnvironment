MassivePixelEnvironment

The MassivePixelEnvironment (MPE) is a Processing library for easily extending sketches to distributed display environments.

Download the library as a .zip file Download this project as a tar.gz file

Welcome to the homepage for the Massive Pixel Environment library for the Processing Programming Language.

You can download the library with the linked icons above. It can be installed by copying the archive contents to your sketchbook's libraries folder.

About

Consider using MPE if you are creating visual designs, visualizations, or interactive installations using more than one computer or driving many displays. MPE abstracts away the distributed nature of the system and lets you focus on visuals.

MPE was developed at the Texas Advanced Computing Center and aims to make it easy to prototype visualizations for otherwise difficult to use distributed systems. MPE is compatible with the lastest Processing versions and supports full compatibility with advanced featuresets of OpenGL. MPE's lead developer is @bmwesting.

In MPE:

  • Each node in a cluster runs an instance of the Processing sketch application
  • The Processing sketch is easily launched across the cluster through the use of a single configuration file
  • Support for all features of Processing is available
